## Visual brief

The BEAM Campus wordmark should express:

- **Academic / infrastructure heritage** — not startup, not Silicon Valley
- **European sensibility** — quiet, confident, serif-leaning display type
- **Reference to the BEAM** — suggesting the Erlang runtime without being literal
- **Resilience / federation** — graph / mesh motif compatible with the Macula mark
